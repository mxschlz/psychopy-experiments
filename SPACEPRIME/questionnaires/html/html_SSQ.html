Sprache, räumliches Hören und Hörqualität (SSQ)

Teil 1: Sprachverstehen

Sie sprechen mit einer anderen Person in einem ruhigen, mit Teppich ausgelegten Raum. Können Sie die andere Person verstehen?

Sie sprechen mit jemandem an einem Ort mit viel Nachhall/Echos, z.B. in einer Kirche oder Bahnhofshalle. Können Sie verstehen, was die andere Person sagt?

Sie sprechen mit einer anderen Person während der Fernseher im selben Raum läuft. Können Sie die andere Person, mit der Sie sprechen, verstehen, ohne den Fernseher leise zu stellen?

Sie hören jemandem zu, der mit Ihnen spricht, während Sie gleichzeitig versuchen, der Nachrichtensendung im Fernsehen zu folgen. Können Sie beiden Personen folgen, was sie sagen?

Sie sind mit einer Gruppe von etwa fünf Personen in einem belebten Restaurant und Sie können jeden in der Gruppe sehen. Können Sie der Unterhaltung folgen?

Sie sind mit einer Gruppe von etwa fünf Personen in einem belebten Restaurant und Sie KÖNNEN NICHT jeden in der Gruppe sehen. Können Sie der Unterhaltung folgen?

Teil 2: Räumliches Hören

Sie sind im Freien an einem Ihnen unbekannten Ort. Sie hören, dass jemand einen Rasenmäher benutzt, aber Sie sehen nicht, wo das ist. Können Sie sofort sagen, woher das Geräusch kommt?

Sie sind in einem Ihnen unbekannten Haus. Es ist ruhig. Sie hören eine Tür schlagen. Können Sie sofort sagen, woher das Geräusch kam?

Sie sitzen am Tisch oder in einer Besprechung mit mehreren Personen. Sie können nicht jeden sehen. Können Sie sagen, wo jemand sitzt, sobald er/sie zu sprechen beginnt?

Sie sitzen zwischen zwei Personen. Eine Person beginnt zu sprechen. Können Sie sofort sagen, ob es die Person zu Ihrer Linken oder Rechten ist, ohne hinzuschauen?

Können Sie anhand des Geräusches sagen, wie weit ein Bus oder Lastwagen entfernt ist?

Können Sie anhand des Geräusches sagen, ob ein Bus oder Lastwagen Ihnen entgegen kommt oder sich entfernt?

Teil 3: Hörqualität

Stellen Sie sich vor, Sie hören zwei verschiedene Geräusche gleichzeitig, z.B. Wasser, das in ein Becken einläuft und gleichzeitig spielt ein Radio. Können Sie beide Geräusche eindeutig voneinander unterscheiden?

Können Sie ähnliche Geräuschquellen eindeutig zuordnen? (Können sie z. B. hören, ob es sich um ein Auto oder einen Bus handelt, bzw. um kochendes Wasser in einem Topf oder Bratgeräuschen in einer Pfanne?)

Empfinden Sie es als einfach verschiedene Leute, die Sie kennen, anhand ihrer Stimme zu erkennen?

Können Sie problemlos die Stimmung einer anderen Person durch den Klang ihrer Stimme beurteilen?

Wenn Sie als Fahrer im Auto fahren, können Sie problemlos verstehen, was eine andere Person sagt, die neben Ihnen sitzt?

Können Sie andere Geräusche mühelos ignorieren, wenn Sie versuchen auf etwas Bestimmtes zu hören?
